# TOOLS — Capability Manifest & Skills Directory

## Skills
- Review the contents of the `src/workspace/skills/[name]/SKILL.md` files, where `[name]` represents the names of individual skills.
- If someone asks, "What are your skills?", respond by listing the folders under `src/workspace/skills/` as skills. Do not refer to them as folders; they are your skills. Answer immediately with `done: true` and a `final_answer`.

## Skill Directory Instructions
- The `src/workspace/skills/` directory contains all registered capability modules.
- Each subdirectory represents a distinct skill with its own documentation and implementation.
- Treat these directories as skills, not folders, when communicating with the user.
